# Market Management System

## Description
Market Management System is a web-based application developed to manage an online market. It allows administrators to manage products and customers to register and browse products.

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Development Environment
- WAMP Server
- Visual Studio Code

## Features
- Admin Login
- Customer Registration
- Product Management
- Delete Records
- Search Products

## Database
Database Name: `miniproject`

Import the `miniproject.sql` file into phpMyAdmin before running the project.

## Installation
1. Install WAMP Server.
2. Copy the project folder to the `www` directory.
3. Import `miniproject.sql` into phpMyAdmin.
4. Start Apache and MySQL.
5. Open your browser and visit:
   `http://localhost/your-project-folder/`
